## Añadir colaboradores en GitHub

1. Repositorio → **Settings** → **Collaborators** (o **Organization** → **Teams**).
2. **Add people** con el correo de la persona a invitar.
3. Rol según necesidad: **Read** si solo revisa o despliega; **Write** / **Maintain** si integra cambios en el código.
